# IROX — Enterprise AI Data Automation Platform

IROX is an elite, high-performance web platform built for enterprise data automation. The architecture abandons traditional SaaS conventions in favor of a cinematic, mathematically driven digital experience that prioritizes industrial authority, cinematic motion, and zero-dependency performance.

## ⚡ Core Architecture

- **Zero-Dependency Motion**: Replaced heavy animation libraries (like GSAP or Framer Motion) with raw mathematics, `IntersectionObserver` hooks, and custom React interpolation to maintain a payload size under the strict 500ms initial load limit.
- **Autonomous Compute Cluster (WebGL)**: A bespoke, native Three.js / React Three Fiber procedural engine utilizing `InstancedMesh`. It renders 300 autonomous glowing server nodes that physically expand, tokenize, and rotate in direct sync with the user's scroll depth.
- **Brutalist Hardware Styling**: Eschews generic rounded aesthetics for sharp, 1px asymmetrical borders, dark-inversion blocks, and kinetic mechanical animations (CSS `spin`, dash-array drawing, and hardware blinking pulses).

## 🛠 Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS (v4) + Custom Procedural CSS variables
- **3D Engine**: Three.js + `@react-three/fiber` + `@react-three/drei`
- **Icons**: Heroicons (inline SVG optimization)

## 🚀 Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
